# Features
Módulos independientes que agrupan lógica y componentes por funcionalidad.

- `scanner/`: Captura de fotos y envío al backend.
- `nutritionist/`: Visualización de macros, calorías y alertas.
- `coach/`: Feedback motivacional y reproducción de voz.
- `dashboard/`: Resumen de metas diarias y progreso.
- `onboarding/`: Registro inicial y datos biométricos.
