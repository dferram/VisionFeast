# API Layer (Routers)
Aquí se definen los endpoints de la aplicación utilizando FastAPI.

- `v1/auth.py`: Autenticación con Google y manejo de sesiones.
- `v1/analysis.py`: Endpoint principal para recibir y procesar imágenes de comida.
- `v1/nutritionist.py`: Endpoints específicos para la lógica de nutrición.
- `v1/coach.py`: Endpoints específicos para el feedback del entrenador.
