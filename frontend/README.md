# Frontend - VisionFeast
Aplicación web construida con React, Tailwind CSS y Vite, siguiendo **Feature-Sliced Design**.

## Estructura
- `src/core`: Configuración global y clientes de API.
- `src/shared`: Componentes UI reutilizables (Botones, inputs, etc).
- `src/features`: Módulos independientes por funcionalidad.
- `src/assets`: Estilos globales e imágenes.
