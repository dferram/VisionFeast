# Service Layer (Business Logic)
El corazón de la aplicación. Aquí reside la "personalidad" de las IAs.

- `ai_gateway.py`: Cliente base para Gemini 1.5 Flash.
- `voice_gateway.py`: Cliente base para ElevenLabs.
- `nutritionist_service.py`: Prompts y lógica de análisis nutricional/alérgenos.
- `coach_service.py`: Prompts y lógica de motivación y entrenamiento.
